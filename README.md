Figure 3 is drawn by Plot_Fig3.m, where the optimized points are generated by Plot_Fig3_leftFIM.m and Plot_Fig3_leftSob.m. 

Figure 4 is drawn by Plot_Fig4.m

Figure 5 is drawn by Plot_Fig5.m, where the optimized points are generated by Plot_Fig5_leftFIM.m and Plot_Fig5_leftSob.m.

Figure 9-10 are drawn by Plot_Fig6.m-Plot_Fig8.m, respectively, using the data generated by Fig9_10_data.m
